Сценарии:

1. Кому написать и что предложить

- Ассистент собирает профиль клиента по clients.notes, заказам и товарам: последний заказ, любимые бренды/категории, средний чек, особенности из заметок. Помогает менеджеру подготовить персональный контакт.

2. Какие товары рискуют закончиться

- Ассистент сравнивает текущие остатки с фактическими продажами. Показывает не просто товары с маленьким остатком, а товары, где остаток мал относительно спроса.

3. Что произошло в продажах за период

- Ассистент делает короткую управленческую сводку: выручка, активные клиенты, ходовые товары, отмены, аномалии и товары, которые требуют внимания из-за связи продаж и остатков.

Курьер не выбран как целевой пользователь MVP, потому что в предоставленных данных нет маршрутов, адресов доставки, времени доставки или статусов отгрузки. Поэтому сценарии сфокусированы на Юле и Ирине, для которых доступны данные по клиентам, заказам и остаткам.
